# 🛒 Retail Sales Analysis Project

## 📌 Project Overview
Analyzed 9,994 rows of Superstore retail sales data to find
key business insights using Python, SQL, and Power BI.
---
## 🛠️ Tools Used
- **Python** — Pandas, NumPy, Matplotlib, Seaborn
- **SQL** — PostgreSQL (PgAdmin)
- **Power BI** — Interactive Dashboard

---

## 📂 Project Structure
- `Superstore_EDA.ipynb` — Python EDA Notebook
- `analysis.sql` — SQL Queries
- `dashboard.png` — Power BI Dashboard Screenshot
- `Sample_-_Superstore.csv` — Dataset

## 🔍 Key Findings
- **Technology** is the highest revenue-generating category
- **West region** is the most profitable region
- **Sales peak in Q4** — November and December
- **Furniture in Central region** is running at a loss
- **Phones and Chairs** are the top selling sub-categories

## Python Analysis
